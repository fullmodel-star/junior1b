# 國一下英語複習（康軒）內部版

國中一年級下學期英語複習 PWA。單字、片語、文法、句型，主打**句子中翻英**。

## 為什麼要密碼

內容取自康軒《英語2 國中1下 學習自修》，屬版權教材，僅供自家孩子複習使用。
題庫在建置時以 **PBKDF2(150k) + AES-256-GCM** 加密內嵌，沒有密碼無法解讀——
即使 `index.html` 放到公開空間，看到的也只是亂碼。

> 密碼採短 PIN（自用取向）。4 位數 PIN 屬「軟鎖」，理論上可離線暴力嘗試；
> 若要真正的強度，改用長密碼重新建置即可（`scripts/build.py` 的 `DEFAULT_PW`）。

## 學習機制（比照「國中2000」）

**單字 / 片語**
- **🃏 字卡**：翻面後三段自評（不會／會了／很簡單）。SRS 盒子 1–6，間隔 `0,0,1,2,4,7,15` 天，
  盒 ≥5 視為精熟。自評「不會」會軟性收進錯題本。
- **📝 測驗**：單字有「英→中／中→英／拼字」三種；片語較長，只有前兩種。

**中翻英**（每課依長度切成 6–14 個小單元，每單元 12–13 句；單元內由短句到長句）
- **🧩 拼句**：點選打散的單字方塊排出句子（含 2 個干擾詞）。方塊不帶標點——
  帶句點的方塊等於告訴孩子那是句尾。
- **⌨️ 打字**：自己輸入。正規化比對（忽略大小寫、標點、撇號、多餘空白）。
- **答錯會告訴他錯在哪**：逐詞比對，指出「少了 the」「拼錯：recieve → receive」「順序不一樣」，
  而不是只丟一個正解。之後才讓孩子自評（一句多譯的情況）；拼句也有「我這樣寫其實也對」。
- 一次做幾題可以自己選（5 / 10 / 整個單元），只做 5 題時優先出還沒練過的。

**文法 / 句型**
- 每課、每個重點都可收放（accordion），讀完點「我讀懂了」打勾，可反悔。
- 顯示每課「讀過 x/y」與總進度，全讀完標記完成。

## 錯題本（單字 / 片語 / 中翻英 三類都收）

答錯即進錯題本，盒子 1–3、間隔 `10 分 → 1 天 → 3 天`。
畢業條件：**每一個曾經答錯的方向**都連對 2 次（例如某字英→中、中→英都錯過，
就要兩個方向各連對 2 次才算學會）。複習頁三類各自可以開始練。

## 建置

需要 Node.js（做加密）與 Python 3。

```bash
python scripts/build.py     # 產生加密 index.html，並自動戳 sw.js 的快取版本
node scripts/verify.js 1019 # 驗解密、防明文外洩、題庫健全性、更新機制
node scripts/smoke.js       # jsdom 實跑 UI（92 項）
node scripts/migrate_test.js# v1→v2 進度遷移
```

**不要手動改 `sw.js` 的 `CACHE`。** 它由 `build.py` 依 `index.html` 的雜湊自動戳上。
若 `sw.js` 位元組沒變，瀏覽器不會安裝新的 Service Worker，已把 App 加到桌面的孩子
就永遠停在第一次快取的舊版，任何內容修正都送不到。`verify.js` 會擋住這種情況。
