# 國一下英語複習（康軒）內部版

國中一年級下學期英語複習 PWA。單字、片語、文法、句型，主打**句子中翻英**。

## 為什麼要密碼

內容取自康軒《英語2 國中1下 學習自修》，屬版權教材，僅供自家孩子複習使用。
題庫在建置時以 **PBKDF2(150k) + AES-256-GCM** 加密內嵌，沒有密碼無法解讀——
即使 `index.html` 放到公開空間，看到的也只是亂碼。

> 密碼採短 PIN（自用取向）。4 位數 PIN 屬「軟鎖」，理論上可離線暴力嘗試；
> 若要真正的強度，改用長密碼重新建置即可（`scripts/build.py` 的 `DEFAULT_PW`）。

## 學習機制（比照「國中2000」）

**單字 / 片語**
- **🃏 字卡**：翻面後三段自評（不會／會了／很簡單）。SRS 盒子 1–6，間隔 `0,0,1,2,4,7,15` 天，
  盒 ≥5 視為精熟。自評「不會」會軟性收進錯題本。
- **📝 測驗**：單字有「英→中／中→英／拼字」三種；片語較長，只有前兩種。

**中翻英**（每課拆成 4 個小單元，約 20–27 句，一次不會太多）
- **🧩 拼句**：點選打散的單字方塊排出句子（含 2 個干擾詞）。自動批改、免拼字。
- **⌨️ 打字**：自己輸入。正規化比對（忽略大小寫、標點、撇號、多餘空白）；
  比對不過時顯示參考答案，讓孩子**自評**（涵蓋一句多譯的情況）。

**文法 / 句型**
- 每課、每個重點都可收放（accordion），讀完點「我讀懂了」打勾，可反悔。
- 顯示每課「讀過 x/y」與總進度，全讀完標記完成。

## 錯題本（單字 / 片語 / 中翻英 三類都收）

答錯即進錯題本，盒子 1–3、間隔 `10 分 → 1 天 → 3 天`。
畢業條件：**每一個曾經答錯的方向**都連對 2 次（例如某字英→中、中→英都錯過，
就要兩個方向各連對 2 次才算學會）。複習頁三類各自可以開始練。

## 建置

需要 Node.js（做加密）與 Python 3。

```bash
python scripts/build.py
```

會讀 `_source/` 的資料，產出加密後的 `index.html`。

## 資料夾

```
_source/            版權明文資料（.gitignore，絕不上傳）
  vocab_phrases.json   單字＋片語
  lesson_1..6.json     文法／句型／中翻英句子池
scripts/
  build.py          組資料 → 呼叫 encrypt.js
  encrypt.js        PBKDF2 + AES-256-GCM 加密
  template.html     App 版面與引擎（無題庫內容）
index.html          產出物：自包含、已加密
```

只有加密後的 `index.html`、圖示、`manifest`/`sw` 會進 repo。
